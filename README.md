# ur-registry-xtz

This repository is the tezos extension of [bc-ur-registry](https://github.com/airgap-it/ur-registry-xtz)

## Installing

To install, run:

```bash
yarn add @airgap/ur-registry-tezos
```

```bash
npm install --save @airgap/ur-registry-tezos
```
